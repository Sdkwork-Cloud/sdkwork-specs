# SDKWork Plugins

Repository or application plugins live under:

```text
.sdkwork/plugins/<plugin-name>/
  .codex-plugin/
    plugin.json
```

Plugin names use lowercase kebab-case. Installable plugins must include `.codex-plugin/plugin.json` and document contributed skills, MCP servers, apps, or scripts.

This repository currently keeps the plugins directory as a standards-compliant placeholder. Do not place local credentials, runtime files, generated SDK output, caches, or logs here.
