# SDKWork Skills

Repository or application skills live under:

```text
.sdkwork/skills/<skill-name>/SKILL.md
```

Skill names use lowercase kebab-case. A skill should describe when to use it, what commands or files it may touch, and which canonical specs it follows. Skills may include `references/`, `scripts/`, or `assets/` when those files are part of the workflow.

Current skills:

- `sdkwork-standards-review` - review workflow for changes to SDKWork standards, repository workspace metadata, API/SDK rules, and related cross-references.
